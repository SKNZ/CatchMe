Associates each (player) token with a list of controls.

Every player token has its own set of controls, composed of 9 keyboard keys, which should form a square. 0 -> Upleft, 1 -> Up, 2 -> Upright 3 -> Left, 4 -> Stay (disabled), 5 -> Right 6 -> Downleft, 7 -> Down, 8 -> Downright

List of all the possible tokens.

A token is an item that occupies a case on the grid, such as nothing, a player, or an obstacle. These, however, do not include the borders. The last token is the empty token.
